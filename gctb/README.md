# GCTB city & town maps (2015–2022)

Per-year world maps of global **city** and **town** extents, one PNG per year
2015–2022, rendered from centroids of the 30 m boundary polygons.

- `gctb_cities_towns_<year>.png` — cities (teal) + towns (amber), WGS-84, global.

**Year range:** the GCTB dataset provides annual boundaries only through **2022**
(and 5-year steps before 2000); there is no 2023–2025 data, so the series stops at 2022.

**Data source (CC-BY-4.0):** Liu, Liangyun; Zhang, Xiao; Bai, Ming. *GCTB: a
high-resolution global annual city and town boundaries dataset (1985–2022) derived
from GLC_FCS30D land cover product.* Zenodo. https://doi.org/10.5281/zenodo.16418717

These urban-boundary maps are an auxiliary layer kept alongside the acoustic
biodiversity analysis; they are not derived from the Xeno-canto pipeline.
